##### Escreve o alfabeto na RAM

* R10: caractere ascii
* R12: endereço da memória

###### Tabela ASCII:  

* 'A' -> 0x41
* [...]
* 'Z' -> 0x5A

###### RAM:

 * 0x0200: 'A'
 * [...]
 * 0x0219: 'Z'
